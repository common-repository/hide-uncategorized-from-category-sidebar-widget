This plugin hides category Uncategorized from the category sidebar widget

== Description ==

If your Woocommerce site contains any products that are not manually assigned to a category, they will be assigned to category Uncategorized. The category Uncategorized will then appear in the category sidebar widget. This plugin hides category Uncategorized from the category sidebar widget.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin through the 'Plugins' menu in WordPress
2. Activate the plugin through the 'Plugins' menu in WordPress
